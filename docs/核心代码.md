## 二、Agent 创建与工厂模式

### 2.1 create_deerflow_agent 函数

**文件**: [factory.py](file:///d:/git_ai/deer-flow/backend/packages/harness/deerflow/agents/factory.py)

**设计理念**: 提供纯参数化的 Agent 创建接口，不依赖配置文件

**核心实现**:

```python
def create_deerflow_agent(
    model: BaseChatModel,
    tools: list[BaseTool] | None = None,
    *,
    system_prompt: str | None = None,
    middleware: list[AgentMiddleware] | None = None,
    features: RuntimeFeatures | None = None,
    extra_middleware: list[AgentMiddleware] | None = None,
    plan_mode: bool = False,
    state_schema: type | None = None,
    checkpointer: BaseCheckpointSaver | None = None,
    name: str = "default",
) -> CompiledStateGraph:
```

**参数验证**:
- 不能同时指定 `middleware` 和 `features`
- 不能在使用 `middleware` 时指定 `extra_middleware`
- `extra_middleware` 必须是 `AgentMiddleware` 实例

**中间件组装策略**:

1. **全接管模式** (`middleware` 参数):
   - 直接使用提供的中间件列表
   - 不进行任何自动组装

2. **特性驱动模式** (`features` 参数):
   - 根据 `RuntimeFeatures` 自动组装中间件链
   - 支持通过 `extra_middleware` 插入自定义中间件

### 2.2 中间件链组装算法

**函数**: `_assemble_from_features`

**固定顺序**:

```
0-2:   Sandbox infrastructure (ThreadData → Uploads → Sandbox)
3:     DanglingToolCallMiddleware (always)
4:     GuardrailMiddleware (guardrail feature)
5:     ToolErrorHandlingMiddleware (always)
6:     SummarizationMiddleware (summarization feature)
7:     TodoMiddleware (plan_mode parameter)
8:     TitleMiddleware (auto_title feature)
9:     MemoryMiddleware (memory feature)
10:    ViewImageMiddleware (vision feature)
11:    SubagentLimitMiddleware (subagent feature)
12:    LoopDetectionMiddleware (always)
13:    ClarificationMiddleware (always last)
```

**特性值处理**:
- `False`: 跳过该中间件
- `True`: 使用内置默认实现
- `AgentMiddleware` 实例: 使用自定义实现

### 2.3 额外中间件插入算法

**函数**: `_insert_extra`

**插入机制**:
- 使用 `@Next` 和 `@Prev` 装饰器指定位置
- `@Next(AnchorClass)`: 插入到锚点类之后
- `@Prev(AnchorClass)`: 插入到锚点类之前

**冲突检测**:
- 同一锚点不能有多个 `@Next`
- 同一锚点不能有多个 `@Prev`
- `@Next` 和 `@Prev` 不能同时指向同一锚点

**迭代插入**:
- 支持外部中间件之间的交叉锚定
- 最多迭代 `len(pending) + 1` 轮
- 无法解析时抛出错误

**不变量保证**:
- `ClarificationMiddleware` 必须始终在最后
- 如果插入操作违反此规则，自动调整

```python
def _filter_messages_for_memory(messages):
    filtered = []
    skip_next_ai = False
    
    for msg in messages:
        if msg.type == "human":
            # 移除上传文件标签
            content = _extract_message_text(msg)
            if "<uploaded_files>" in content:
                stripped = _UPLOAD_BLOCK_RE.sub("", content).strip()
                if not stripped:
                    skip_next_ai = True
                    continue
                # 重建消息
                clean_msg = copy(msg)
                clean_msg.content = stripped
                filtered.append(clean_msg)
            else:
                filtered.append(msg)
        
        elif msg.type == "ai":
            # 只保留无工具调用的 AI 消息
            if not msg.tool_calls and not skip_next_ai:
                filtered.append(msg)
            skip_next_ai = False
    
    return filtered
```

### 4.2 LoopDetectionMiddleware

**文件**: [loop_detection_middleware.py](file:///d:/git_ai/deer-flow/backend/packages/harness/deerflow/agents/middlewares/loop_detection_middleware.py)

**核心算法**:

#### 工具调用哈希

```python
def _hash_tool_calls(tool_calls):
    # 1. 规范化工具调用
    normalized = [
        {"name": tc["name"], "args": tc["args"]}
        for tc in tool_calls
    ]
    
    # 2. 排序以确保顺序无关
    normalized.sort(key=lambda tc: (
        tc["name"],
        json.dumps(tc["args"], sort_keys=True)
    ))
    
    # 3. 生成哈希
    blob = json.dumps(normalized, sort_keys=True)
    return hashlib.md5(blob.encode()).hexdigest()[:12]
```

#### 循环检测

```python
def _track_and_check(self, state, runtime):
    # 1. 获取最后的 AI 消息
    last_msg = messages[-1]
    if last_msg.type != "ai":
        return None, False
    
    tool_calls = last_msg.tool_calls
    if not tool_calls:
        return None, False
    
    # 2. 计算哈希
    call_hash = _hash_tool_calls(tool_calls)
    
    # 3. 更新历史
    with self._lock:
        history = self._history[thread_id]
        history.append(call_hash)
        
        # 限制窗口大小
        if len(history) > self.window_size:
            history[:] = history[-self.window_size:]
        
        # 4. 统计出现次数
        count = history.count(call_hash)
        
        # 5. 判断是否需要干预
        if count >= self.hard_limit:
            return _HARD_STOP_MSG, True
        
        if count >= self.warn_threshold:
            if call_hash not in self._warned[thread_id]:
                self._warned[thread_id].add(call_hash)
                return _WARNING_MSG, False
    
    return None, False
```

#### 强制中断

```python
def _apply(self, state, runtime):
    warning, hard_stop = self._track_and_check(state, runtime)
    
    if hard_stop:
        # 移除工具调用，强制文本输出
        last_msg = messages[-1]
        stripped_msg = last_msg.model_copy(update={
            "tool_calls": [],
            "content": last_msg.content + f"\n\n{_HARD_STOP_MSG}"
        })
        return {"messages": [stripped_msg]}
    
    if warning:
        # 注入警告消息
        return {"messages": [HumanMessage(content=warning)]}
    
    return None
```

```python
class DeferredToolRegistry:
    def __init__(self):
        self._tools: dict[str, BaseTool] = {}
    
    def register(self, tool: BaseTool):
        self._tools[tool.name] = tool
    
    def get(self, name: str) -> BaseTool | None:
        return self._tools.get(name)
    
    def search(self, query: str) -> list[BaseTool]:
        # 简单的关键词匹配
        results = []
        for tool in self._tools.values():
            if self._matches(query, tool):
                results.append(tool)
        return results
```

```python
async def create_or_reject(
    self, thread_id, assistant_id, *,
    on_disconnect, metadata, kwargs, multitask_strategy
) -> RunRecord:
    async with self._lock:
        # 检查现有运行
        inflight = [r for r in self._runs.values()
                    if r.thread_id == thread_id
                    and r.status in (RunStatus.pending, RunStatus.running)]
        
        # 拒绝策略
        if multitask_strategy == "reject" and inflight:
            raise ConflictError(f"Thread {thread_id} already has active run")

        if multitask_strategy in ("interrupt", "rollback") and inflight:
            for r in inflight:
                r.abort_action = multitask_strategy
                r.abort_event.set()
                if r.task and not r.task.done():
                    r.task.cancel()
                r.status = RunStatus.interrupted

        self._runs[run_id] = record
        return record
```

**取消运行**:

```python
async def cancel(self, run_id: str, *, action: str = "interrupt") -> bool:
    async with self._lock:
        record = self._runs.get(run_id)
        if not record:
            return False
        
        if record.status not in (RunStatus.pending, RunStatus.running):
            return False
        
        # 设置中止事件
        record.abort_action = action
        record.abort_event.set()
        
        # 取消任务
        if record.task and not record.task.done():
            record.task.cancel()
        
        record.status = RunStatus.interrupted
        return True
```

```python
    if thread_id is None:
        thread_id = str(uuid.uuid4())
    
    config = self._get_runnable_config(thread_id, **kwargs)
    self._ensure_agent(config)
    
    state = {"messages": [HumanMessage(content=message)]}
    context = {"thread_id": thread_id}
    
    seen_ids = set()
    cumulative_usage = {"input_tokens": 0, "output_tokens": 0}
    
    for chunk in self._agent.stream(state, config=config, context=context, stream_mode="values"):
        messages = chunk.get("messages", [])
        
        for msg in messages:
            msg_id = getattr(msg, "id", None)
            if msg_id and msg_id in seen_ids:
                continue
            if msg_id:
                seen_ids.add(msg_id)
            
            # 处理不同消息类型
            if isinstance(msg, AIMessage):
                # 追踪 token 使用
                usage = getattr(msg, "usage_metadata", None)
                if usage:
                    cumulative_usage["input_tokens"] += usage.get("input_tokens", 0)
                    cumulative_usage["output_tokens"] += usage.get("output_tokens", 0)
                
                # 发送事件
                if msg.tool_calls:
                    yield StreamEvent(type="messages-tuple", data={...})
                if msg.content:
                    yield StreamEvent(type="messages-tuple", data={...})
            
            elif isinstance(msg, ToolMessage):
                yield StreamEvent(type="messages-tuple", data={...})
        
        # 发送状态快照
        yield StreamEvent(type="values", data={...})
    
    # 发送结束事件
    yield StreamEvent(type="end", data={"usage": cumulative_usage})
```
